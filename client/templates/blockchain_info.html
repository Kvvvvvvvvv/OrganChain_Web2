How to View Transaction Details in Ganache

To view detailed transaction information in Ganache:

  1. Open Ganache and go to the "Transactions" tab
  2. Find the transaction with the hash shown above
  3. Click on the transaction to view details
  4. In the transaction details, go to the "Event Logs" section
  5. You will see detailed information about the match including:
    • Donor name and age
    • Patient name and age
    • Hospital information
    • Organ type (encrypted)
    • Blood type (encrypted)
    • Match date

Blockchain Benefits

  • Transparency: All matches are recorded on the blockchain
  • Immutability: Match records cannot be altered or deleted
  • Security: Sensitive data is encrypted
  • Auditability: All transactions can be traced and verified
